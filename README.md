
# Color Scheme Switcher

A simple and interactive web application that allows users to switch between different color schemes for the background. This project demonstrates basic DOM manipulation and event handling in JavaScript, along with clean and responsive UI using CSS.

## Features
- Switch between multiple color schemes with a single click
- Responsive design for desktop and mobile
- Easy to customize and extend
- Minimal and clean user interface
  

## Getting Started

### Prerequisites
- Modern web browser (Chrome, Firefox, Edge, etc.)

### Installation
1. Clone or download this repository:
	```bash
	git clone https://github.com/01Adityasoni/Color-Scheme-Switcher-.git
	```
2. Open the folder in your code editor or directly in your browser.
3. Open `index.html` to view the app.

## Usage
- Click on any color button to change the background color of the page.
- You can add more colors by editing the `index.js` , `index.html` and `styles.css` files.

## Project Structure
```
├── index.html      # Main HTML file
├── index.js        # JavaScript logic for color switching
├── styles.css      # Styling for the app
└── README.md       # Project documentation
```

## Customization
- To add more colors, update the conditions in `index.js` , `index.html` and add corresponding styles in `styles.css`.
- You can modify the layout and design as per your preference.
